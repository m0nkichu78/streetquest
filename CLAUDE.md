# StreetQuest — Instructions pour Claude

## Icônes
- Toujours utiliser **@heroicons/react** pour toutes les icônes.
- Importer uniquement depuis `@heroicons/react/24/outline` ou `@heroicons/react/24/solid` selon le contexte (outline pour les icônes neutres/secondaires, solid pour les icônes actives/primaires).
- Ne jamais utiliser d'autres librairies d'icônes (lucide-react, react-icons, etc.).

Exemple d'import :
```tsx
import { MagnifyingGlassIcon } from "@heroicons/react/24/outline";
import { CheckCircleIcon } from "@heroicons/react/24/solid";
```
